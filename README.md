# Finance Tracker

A CLI tool (with future web dashboard) to track personal expenses and income.

## Project Status

**Phase 1 (Current):** CLI MVP  
- [ ] Add transactions  
- [ ] List transactions  
- [ ] View balance  
- [ ] Export to CSV  

**Phase 2:** REST API  
**Phase 3:** React Dashboard  

---

## Tech Stack

- **Backend:** Node.js, SQLite
- **CLI:** Native Node.js
- **Future:** Express (API), React (Dashboard)

---

## Setup
```bash
cd backend
npm install
node src/cli/index.js --help
```
